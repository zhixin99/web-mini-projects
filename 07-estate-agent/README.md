This is a simple estate agent website UX. 

It renders out a card for each of the properties in the propertyForSaleArr array. Each card have an image, a property location, a price, a comment and the TOTAL property size in square metres.

If no array of properties is passed to getPropertyHtml, the placeholder property stored in placeholderPropertyObj will be rendered instead.

## Key learning points
- import/export
- .map()
- .join()
- Object destructuring
- .reduce()
- Default parameters
