This is meme picker that will display a cat meme based on the selected emotion. 

This is the first time that I learnt how to consolidate the data in a seperate js file and export/import the data. 

Also, I have learnt 2 others ways to take control of the HTML element, using `querySelector` & `getElementByClassName`.
