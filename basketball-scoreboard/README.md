This is a basketball scoreboard app built using JavaScript, HTML, and CSS for tracking and displaying basketball game scores in real-time. It allows users to start, pause and resume the game.  

The key learning points from this sole project: 
* script tag
* variables
* numbers & strings
* function
* the DOM
* getElementById
* addEventListener
* textContent
* disabled
* setInterval & clearInterval
* @font-face & font-family

Display: 
* The new game button is default to be disabled. 
![The new game button is default to be disabled](images/default-page.png)

* After the game starts, start button is changed to pause button, and new game button is allowed. 
![After the game starts, start button is changed to pause button, and new game button is allowed. ](images/pause-new-game-enable.png)

* Click the score buttons to add the score. 
![Click the score buttons to add the score. ](images/add-score.png)

* When the game ends, there will be a game over alarm. 
![When the game ends, there will be a game over alarm. ](images/game-over-alarm.png)

* New game is waited to be started.
![New game is waited to be started.](images/pause-disable.png)
