This is a password generator app built using JavaScript, HTML, and CSS. It generates 2 random passwords when the user clicks the button. 
* It allows the users to set the password length. (Default to be 15 characters long).
* It has a "copy-on-click" feature.
* It has toggle options where user can modify whether they want symbols and or numbers in their passwords or not.

## Key points
* if condition
* logic operators
* loops 
* Math.
* slider
* toggle button
* setTimeout
* navigator.clipboard.writeText
